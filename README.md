# Oficina de carros

    Sistema de controle de revisões de carros em uma oficina.


# Tecnologias usadas

    Spring - Java


# Dependências

    Spring Web
    Spring Data JPA
    Lombok
    Spring Boot DevTools
    H2 Database


# Execução

    [LINUX] Execute o comando para limpar e reconstruir o projeto spring:

    mvn clean package
    mvn dependency:tree


    [LINUX] Execute o comando para rodar o projeto:

    mvn spring-boot:run
